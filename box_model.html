Boks modellen

Boksmodellen består av innholdet, padding, border og margin.

Margin

Margin er luft rundt innholdet. Denne luften er usynlig

Padding

Rommet mellom innholdet og grensen.

Border

Den angir verdiene til kantbredde, kantstil og kantfarge.

Selector i CSS

Delen av et CSS-regelsett som faktisk velger innholdet du vil style.

Value i CSS

En formell grammatikk, brukes til å definere settet med gyldige verdier for en CSS-egenskap eller -funksjon.

Attribute i CSS

Det er mulig å utforme HTML-elementer som har spesifikke egenskaper eller egenskapsverdier.

"P" [paragraph]

Gir deg mulighet å lage tekst bokser og er en type selector for HTML elementet eller node type.

"H1"

Er en HTML-tag som indikerer en overskrift på et nettsted.
